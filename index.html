2DGalaxyWorld

2DGalaxyWorld is a free 2D game inspired by Minecraft made by JeffTheMasterr. You can build and destroy structures, and simply have fun when playing around with this game.

Editions

This game is available on some coding project websites.

Flowlab game (on flowlab.io)
Scratch game (on scratch.mit.edu)
Turbowarp version of the Scratch game (on turbowarp.org)
